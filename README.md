## KMP
2020년 3월 18일 수요일

- 시간 복잡도 : O(N + M)
- 문자열 검색을 보다 효율적으로 하기위한 알고리즘
- 찾고자 하는 문자열이 문자열 안에 있는지를 빠르게 검색
- **실패함수**를 써서 시간을 효율을 높임(접두사와 중복되지 않는 지점부터 검색을 실시함)
- **찾기 즉, 탐색알고리즘이라고 생각**
- 환형으로 이어지는 배열에서 탐색시 좋음
- [String_Search.md](./String_Search.md)
